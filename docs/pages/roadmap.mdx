# ロードマップ

現在は、必要最小限の機能のみを提供した α 版をリリースしています。今後は、以下を目安に機能拡充を行っていきます。

## 現在

現在取り組んでいることと、最近リリースされた機能。詳細は [GitHub](https://github.com/superfastcms/superfast/releases) から入手できます。

- α 版のリリース
- 画像のアップロード
- セレクト・チェックボックスなどテキストフィールド以外のコンテンツタイプの拡充
- 関連テーブルへの対応
- 検索機能
- その他多数...

## 次に取り組むこと

次に私たちが計画していること。

- MySQL、PostgreSQL をはじめとした対応 Database の拡張
- Superfast と組み合わせて使えるブログスターターキットの提供

2023 年夏を予定。
