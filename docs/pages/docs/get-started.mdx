# Get Started - 始めましょう

Collections を使い始める最も簡単な方法は、create-collections-app パッケージを使用することです。

```sh
npx create-collections-app my-app
```

[http://localhost:4000/admin](http://localhost:4000/admin) を開くと、実行中のアプリが表示されます。本番の準備ができたら `npm run build` `npm run start` を実行するだけです。

## 技術スタック

- セルフホスト型 - データの所有権をあなたへ!
- Database - RDB(MySQL, PostgreSQL, SQLite)
- バックエンド - Node.js, Express
- 管理機能 - React, MUI

Enjoy!!!
