# Migration from WordPress

> Collections allows you to perform content management from scratch, or you can use the import feature to carry over data from WordPress.

## 1. Exporting Contents from WordPress

To begin, export your content from WordPress. Select `Tools` > `Export` > `All content` to perform the download.

## 2. Importing exported files

If the data modal is not yet registered, you will see this initial screen. Click `Import from WordPress XML` to display the modal for import.

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/create-first-model.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

Drag and drop the file you just downloaded.

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/create-first-model-modal.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

If the import is successful, you should see your WordPress content in Content Management.
