# データモデル

> Collections データモデルは、あらゆる SQL データベースの構成と管理をノーコードで提供します。テーブル、カラム、リレーションシップの構成だけでなく、ユーザーがどのようにデータを表示し、操作するかも制御できます。

## データベース

Collections は SQL データベースラッパーです。データベースとは、コンピュータに構造化された方法で保存されたデータの集合のことで、構造化する具体的な方法はデータモデルと呼ばれます。

## リレーションシップ

Collections では、テーブルをリレーションリンクする方法として1種類提供しています。

- **1対N (One to Many)** — データテーブルの各行は、別のデータテーブルの多くの行にリンクできる。
  注：データモデルでは、Many-to-One リレーションシップと One-to-Many リレーションシップは同じ意味です。
