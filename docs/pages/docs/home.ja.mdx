# Collections へようこそ

> これらのドキュメントは、Collections をユニークなものとするコアコンセプトについて説明します。

## Collections とは？

Collections は、AI 検索向けに設計されたヘッドレス CMS です。SEO 対策（Google E-E-A-T、構造化データ）、多言語化、AI による添削を活用し、効果的なコンテンツを作成できます。

## ヘッドレスとは？

フロントエンドとバックエンドを分離するアーキテクチャのことです。特徴的なのは、一体型の CMS とは異なりコンテンツを配信する画面がないことです。そのため、ヘッドレス CMS が提供する API と独自のフロントエンドを組み合わせて優れたユーザー体験を提供することができます。

## Collections を使うメリット

Collections に入稿したコンテンツは、様々なサイトに掲載することができます。既存の Web サイトだけでなく [WordPressに埋め込む](https://collections.dev/ja/docs/use-cases/embed-wordpress) こともできるため、大掛かりな移行を伴うことなくコンテンツを最適化できるのが最大のメリットです。
