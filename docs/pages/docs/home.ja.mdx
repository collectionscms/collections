# Collections へようこそ

> これらのドキュメントは、Collections をユニークなものとするコアコンセプトについて説明します。

## Collections とは？

Collections は、ブログ・お知らせ・通知など、Webサイトに動的な多言語コンテンツを組み込めるヘッドレスCMSです。入稿画面から記事を作成して、自動翻訳と AI による SEO 生成で、既存のサイトを**グローバルな多言語サイトに進化**させることができます。

## ヘッドレスとは？

フロントエンドとバックエンドを分離するアーキテクチャのことです。特徴的なのは、一体型の CMS とは異なりコンテンツを配信する画面がないことです。そのため、ヘッドレスCMSが提供する API と独自のフロントエンドを組み合わせて高いユーザー体験のサイトを提供することができます。

## Collections を使うメリット

Collections で作成したコンテンツは、既存サイトや WordPress にも埋め込むことができます。そのため、大掛かりな移行を行うことなく多言語化が可能です。
