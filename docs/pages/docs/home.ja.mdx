# Collections へようこそ

> これらのドキュメントは、Collections をユニークなものとするコアコンセプトについて説明します。

## Collections とは？

Collections は、ブログ・お知らせ・通知など、Webサイトに動的な多言語コンテンツを組み込めるヘッドレスCMSです。入稿画面から記事を作成して、自動翻訳と AI による SEO 生成で既存のサイトを**グローバルな多言語サイトに進化**させることができます。

## ヘッドレスとは？

フロントエンドとバックエンドを分離するアーキテクチャのことです。特徴的なのは、一体型の CMS とは異なりコンテンツを配信する画面がないことです。そのため、ヘッドレスCMSが提供する API と独自のフロントエンドを組み合わせてユーザー体験の良いサイトを提供することができます。

## Collections を使うメリット

Collections で作成したコンテンツは、既存サイトや[WordPressに埋め込む](https://collections.dev/ja/docs/use-cases/embed-wordpress)ことができます。そのため、大掛かりな移行を行うことなく手軽にサイトの多言語化が可能です。
