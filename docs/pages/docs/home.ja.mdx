# Collections へようこそ

> これらのドキュメントは、Collections をユニークなものとするコアコンセプトについて説明します。

## Collections とは？

Collections は、ブログ・お知らせ・通知など、Webサイトに動的な多言語コンテンツを組み込めるヘッドレスCMSです。入稿画面から記事を作成して、自動翻訳と AI による SEO 生成で、既存のサイトを手軽に**グローバルな多言語サイトに進化**させることができます。

## ヘッドレスとは？

フロントエンドとバックエンドを分離するアーキテクチャのことです。特徴的なのは、WordPress や WIX といった一体型の CMS とは異なりコンテンツを配信する画面がないことです。そのためコンテンツ制作者は、ヘッドレスCMSが提供する API と独自のフロントエンドを組み合わせてサイトを構築する必要があります。

## ヘッドレスのメリット

独自のフロントエンドにより、ユーザー体験の良いサイトを提供することができます。さらに、Collections と組み合わせることで、多言語サイトをイチから構築したり**既存サイトの多言語化**を行うことができます！
