# ユーザーの招待

> このガイドでは、プロジェクトに新たにユーザーを招待する手順を説明します。

<Steps>
### ユーザー作成画面に進む

「Create New / 新規登録」ボタンをクリックして作成画面に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/invite-user/invite-step-1.png" />

### ユーザー情報を入力する

招待するユーザーのメールアドレスと、そのユーザーに付与するロールを選択します。新しいロールの作成手順は[こちら](/docs/app-guides/role-creation)を参照してください。

<Image imageUrl="https://cdn.collections.dev/docs/invite-user/invite-step-2.png" />

### ユーザーの招待完了

ユーザー一覧に新しく招待したユーザーが表示されていることを確認してください。

<Image imageUrl="https://cdn.collections.dev/docs/invite-user/invite-step-3.png" />

</Steps>
