# Add Multilingual Content

> This guide explains how to add multilingual content and provides instructions for translating from the source language.

<Steps>
### Proceed to the post creation screen

In the post creation screen, click the "Add" button in the language selection section.

<Image imageUrl="https://cdn.collections.dev/docs/add-multilingual-content/content-step-1.png" />

### Add language

Add the language in which you want to create multilingual content.

<Image imageUrl="https://cdn.collections.dev/docs/add-multilingual-content/content-step-2.png" />

### Translate from source language

Click the "I do" button to translate the post from the source language.

<Image imageUrl="https://cdn.collections.dev/docs/add-multilingual-content/content-step-3.png" />

### Check the translation results

Once the translation is complete, it will be reflected in the content.

<Image imageUrl="https://cdn.collections.dev/docs/add-multilingual-content/content-step-4.png" />

</Steps>
