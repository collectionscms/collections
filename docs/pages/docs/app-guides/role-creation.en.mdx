# Role Creation

> Every project comes with an admin role by default. This guide explains how to create a new role with more restricted permissions.

<Steps>
### Proceed to role creation page

Click the "Create New" button to proceed to the creation page.

<Image imageUrl="https://cdn.collections.dev/docs/role-creation/role-step-1.png" />

### Enter role information

Enter the role name and description, then select the permissions. In this case, we'll create a role with permissions to create, edit, and delete posts.

<Image imageUrl="https://cdn.collections.dev/docs/role-creation/role-step-2.png" />

### Role creation complete

That's it! The role has been created. Please check the roles list to ensure that the new role is displayed.

</Steps>
