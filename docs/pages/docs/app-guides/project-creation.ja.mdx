# プロジェクトの作成

> このガイドでは、はじめて Collections のプロジェクトを作成する手順を説明します。

<Steps>
### プロジェクト作成画面に進む

サインアップ完了後「Create New / 新規登録」ボタンをクリックしてプロジェク作成画面に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/project-registration/pj-step-1.1.png" />

### プロジェクト情報を入力する

プロジェクトの名前とIDを入力します。プロジェクトIDは、API の URL としても使用され、プロジェクトを特定する一意の識別子になります。

<Image imageUrl="https://cdn.collections.dev/docs/project-registration/pj-step-2.1.png" />

### 記事の原文言語を選択する

この言語は、記事作成時のデフォルト言語となり、他言語に翻訳する際の原文言語にもなります。この設定は後から変更可能です。

<Image imageUrl="https://cdn.collections.dev/docs/project-registration/pj-step-3.1.png" />

### プロジェクトの作成完了

以上で作成完了です。ホーム画面に進んで、プロジェクトが作成されていることを確認してください。

<Image imageUrl="https://cdn.collections.dev/docs/project-registration/pj-step-4.1.png" />

</Steps>
