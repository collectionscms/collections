# Creating and Publishing posts

> Here's a guide to creating and publishing post in Collections.

<Steps>
### Proceed to the post creation page

Proceed to the create post page from the list view.

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-1.png" />

### Complete the post and proceed to publish settings

Enter the title and text to complete the post and proceed to the publication settings.

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-2.png" />

### Edit public settings

The description of each field is as follows.

| Field           | Description                                                                                                                   |
| :-------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| Status          | Select either "Review" or "Publish." Some roles may only be permitted to select "Review."                                     |
| Post slug       | This is the identifier used when retrieving data via the API. It can be specified individually for each multilingual content. |
| Tags            | Multiple tags can be assigned.                                                                                                |
| SEO title       | Enter a title for SEO purposes. It can also be automatically generated by AI based on the content.                            |
| SEO description | Enter a description for SEO purposes. Like the title, it can also be automatically generated by AI based on the content.      |

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-3.png" />

### Publish post

Finally, click the "Publish" button to make your post live.

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-4.png" />

</Steps>
