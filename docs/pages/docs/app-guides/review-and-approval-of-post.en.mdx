# Review and Approval of Post

> This guide explains the steps for creating an article, reviewing it, approving it, and finally publishing it.

<Steps>
### Proceed to the post creation page

Proceed to the create post page from the list view.

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-1.png" />

### Complete the post and proceed to publish settings

Enter the title and text to complete the post and proceed to the publication settings.

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-2.png" />

### Request a review

Select the status as 'Review' enter your comments, and then request the review.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-1.png" />

Returning to the list, you will see that the post status is now set to 'Review'.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-1-2.png" />

### Go to review list

Users with administrator or review viewing permissions will see the review request listed once they log in.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-2.1.png" />

In the review details page, you can check the comments and the post, and decide whether to approve it.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-3.png" />

### Approval

If there are no issues with the post, click the 'Approve and Publish' button to make it live.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-4.png" />

The review is complete. When you return to the post list, you will see that the post status has changed to 'Published'.

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-4-2.png" />

</Steps>
