# 記事のレビューと承認

> このガイドでは、記事を作成してレビュー、承認、公開するまでの手順を説明します。

<Steps>
### 記事の作成画面に進む

一覧から記事の作成画面に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-1.png" />

### 記事を完成させて公開設定に進む

タイトルや本文を入力して記事を完成させ、公開設定に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-2.png" />

### レビューを依頼する

ステータスは「Review / レビュー」を選択し、コメントを入力した上でレビューを依頼します。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-1.png" />

一覧に戻ると、記事のステータスが「Review / レビュー」になっているのが確認できます。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-1-2.png" />

### レビュー一覧に進む

管理者やレビューの閲覧権限を持つユーザーでログインすると、先ほど依頼したレビューが一覧に表示されます。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-2.1.png" />

レビュー詳細画面でコメントと記事を確認し、承認するかを決定します。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-3.png" />

### 承認する

記事に問題がなければ「Approve and publish / 承認して公開」ボタンをクリックして記事を公開します。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-4.png" />

以上でレビューが完了です。記事一覧に戻ると、記事のステータスが「Published / 公開中」になっているのが確認できます。

<Image imageUrl="https://cdn.collections.dev/docs/review-and-approval-of-post/review-step-4-2.png" />

</Steps>
