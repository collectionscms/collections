# 記事の作成と公開

> このガイドでは、記事を作成して公開する手順を説明します。

<Steps>
### 記事の作成画面に進む

一覧から記事の作成画面に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-1.png" />

### 記事を完成させて公開設定に進む

タイトルや本文を入力して記事を完成させ、公開設定に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-2.png" />

### 公開設定を編集する

各フィールドの説明は、以下の通りです。

| フィールド            | 説明                                                                                                       |
| :-------------------- | :--------------------------------------------------------------------------------------------------------- |
| ステータス            | 「レビュー」または「公開」を選択します。ロールによっては、レビューのみしか許可されていない場合があります。 |
| 記事のスラッグ        | API で取得する際の識別子になります。多言語コンテンツごとに指定することが可能です。                         |
| タグ                  | 複数のタグを付与することができます。                                                                       |
| SEOタイトル           | SEO向けのタイトルを入力します。本文をもとに AI による自動生成も可能です。                                  |
| SEOディスクリプション | SEO向けのディスクリプションを入力します。タイトルと同様、AI による自動生成も可能です。                     |

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-3.png" />

### 記事を公開する

最後に「Publish / 公開する」ボタンをクリックして記事を公開します。

<Image imageUrl="https://cdn.collections.dev/docs/creating-and-publishing-posts/post-step-4.png" />

</Steps>
