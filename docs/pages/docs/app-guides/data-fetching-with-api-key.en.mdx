# Data fetching with API key

> Here's a guide on how to use an API key to fetching published posts.

<Steps>
### Create an API key

First, create an API key to fetching data. Click the "Create New" button to proceed to the creation page.

<Image imageUrl="https://cdn.collections.dev/docs/data-fetching-with-api-key/key-step-2.png" />

### Enter API information

Enter a name for the API key and select the permissions. This will create the API key.

<Image imageUrl="https://cdn.collections.dev/docs/data-fetching-with-api-key/key-step-3.png" />

### Open API Preview

Next, open the API preview from the post list.

<Image imageUrl="https://cdn.collections.dev/docs/data-fetching-with-api-key/data-fetch-1.png" />

### Send request

In the API preview, you can send a data retrieval request using the API key you created earlier.

<Image imageUrl="https://cdn.collections.dev/docs/data-fetching-with-api-key/data-fetch-2.png" />

### Copy cURL command

Clicking the copy button allows you to copy a cURL command to send the same request.
For more details on API endpoints and parameters, please refer to the [documentation](/reference/api).

<Image imageUrl="https://cdn.collections.dev/docs/data-fetching-with-api-key/data-fetch-3.png" />

</Steps>
