# ロールの作成

> すべてのプロジェクトには、はじめから管理者ロールが設定されています。このガイドでは、それよりも制限された権限を持つ新しいロールを作成する手順を説明します。

<Steps>
### Proceed to the role creation screen

「Create New / 新規登録」ボタンをクリックして作成画面に進みます。

<Image imageUrl="https://cdn.collections.dev/docs/role-creation/role-step-1.png" />

### ロール情報を入力する

ロールの名前、説明を入力し、パーミッションを選択します。今回は、記事の作成、編集、削除権限を持つロールを作成します。

<Image imageUrl="https://cdn.collections.dev/docs/role-creation/role-step-2.png" />

### ロールの作成完了

以上で作成完了です。ロール一覧から新しく作成したロールが表示されていることを確認してください。

</Steps>
