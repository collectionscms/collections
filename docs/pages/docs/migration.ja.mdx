# WordPress からの移行

> Collections では、はじめからコンテンツ管理を実行するほかに、インポート機能を使って WordPress からデータを引き継ぐこともできます。

## 1. WordPress からコンテンツをエクスポートする

まずはじめに、WordPress からコンテンツをエクスポートします。`Tools` > `Export` > `All content` を選択してダウンロードを実行します。

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/wp-1.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

## 2. エクスポートしたファイルをインポートする

データモーデルが未登録の場合、このような初期画面が表示されます。`Import from WordPress XML` をクリックして取り込み用のモーダルを表示します。

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/create-first-model.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

さきほどダウンロードしたファイルをドラッグ＆ドロップしてみましょう。

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/create-first-model-modal.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

取り込みが成功すると、コンテンツ管理に WordPress のコンテンツが表示されるはずです。
