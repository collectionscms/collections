# セレクション

> インターフェイスは、ユーザーがコンテンツページのフィールドを操作する方法です。これらは標準的な選択インターフェースです。

## セレクトドロップダウン

<p class="nx-mt-6 nx-leading-7 first:nx-mt-0">
  <img
    src="https://cdn.collections.dev/docs/dropdown.png"
    style={{ boxShadow: '0 0 6px #17294033', borderRadius: '4px' }}
  />
</p>

選択肢のリストから値を選択できるようにするフィールド。

- **タイプ**: `String`
- **選択肢**: ドロップダウンのオプション
  - **ラベル**: ドロップダウンに表示されるラベル
  - **値**: 保存される生のデータ
