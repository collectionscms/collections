> インターフェイスは、ユーザーがコンテンツページのフィールドを操作する方法です。これらはマークダウン記法のテキストエディタで、GitHub Flavored Markdown にも対応しています。

- **タイプ**: `Text`

## 見出し

Atx 形式のヘッダをサポートしています。行頭に1-6個のハッシュ文字を使用し、ヘッダレベル1-6に対応します。

```
# this is head1
## this is head2
### this is head3
#### this is head4
##### this is head5
###### this is head6
```

## 箇条書き

### リスト

ハイフン（-）、プラス（+）、アスタリスク（\*）とスペースを入れることで、リストを表示することができます。また、それら記号の前に
`tab` またはスペース2つを入れると、ネストしたリストになります。

```
- リスト1
  - リスト1-1
- リスト2
  - リスト2-1
- リスト3
```

- リスト1
  - リスト1-1
- リスト2
  - リスト2-1
- リスト3

### 番号付きリスト

```
1. リスト1
2. リスト2
3. リスト3
```

1. リスト1
2. リスト2
3. リスト3

### アスタリスク

```
* to do
* done
```

- to do
- done

## 引用

大なり記号を使用すると、テキストを引用符のようにフォーマットすることができます。

```
> シンプルであることを保て。
```

> シンプルであることを保て。

## コードスニペット

### インラインコード

インラインのコードスニペットには、グレイヴ・アクセントキー（`）を使用します。

```
React as a `script` tag
```

React as a `script` tag

### コードブロック

3つのグレイヴ・アクセントキー（`）で囲むことで、コンテンツ内に大きなコードブロックを表示することができます。

````
```
import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
return <div>Hello {name}</div>;
}
```
````

```
import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
  return <div>Hello {name}</div>;
}
```

また、コードハイライトにも対応しています。次のように JavaScript のコードブロックに、シンタックスハイライトを追加することができます。

```
~~~js
import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
  return <div>Hello {name}</div>;
}
~~~
```

```js
import { createRoot } from 'react-dom/client';

function HelloMessage({ name }) {
  return <div>Hello {name}</div>;
}
```

## テキストフォーマット

### 強調

テキストをダブルアスタリスク（\*\*）で囲むと太字になり、アスタリスク1文字で囲むと、斜体になります。また、それを組み合わせて表示することもできます。

```
**Strong Text**
*Italic text*
***Italic + Strong text***
```

**Strong Text** <br/>
_Italic text_ <br/>
**_Italic + Strong text_**

## 取り消し線

テキストをチルダ（~）1文字で囲むと、取り消し線が表示されます。

```
Reactは、2016年から~5年~6年連続で利用率の高いライブラリ
```

Reactは、2016年から~5年~6年連続で利用率の高いライブラリ

## 画像

`![Text](https://xxxx)` の構文を使用して、画像を表示することができます。

```
![アイコン](https://cdn.collections.dev/docs/icon.png)
```

![アイコン](https://cdn.collections.dev/docs/icon.png)

## リンク

`[Text](https://xxxx)` の構文を使用して、インラインリンクを含めることができます。

```
[Collections](https://collections.dev/) は最高のデベロッパーエクスペリエンスを提供する。
```

[Collections](https://collections.dev/) は最高のデベロッパーエクスペリエンスを提供する。

## 区切り線

アスタリスク、アンダースコア、ハイフンを3つ以上連続して記述することで区切り線を表示することができます。

```
***
___
---
```

---

---

---
