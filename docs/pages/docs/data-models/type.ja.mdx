# データの型

> データの型には、リスト型とシングルオブジェクト型があります。どちらを選択しても、後で変更することが可能です。

## リスト型

リスト型は、複数のデータをまとめて扱うことができます。レスポンスは以下のような配列で返されます。

```json
{
  "data": [
    {
      "id": "1",
      "name": "John Doe",
      "age": 30
    },
    {
      "id": "2",
      "name": "Jane Doe",
      "age": 24
    }
  ]
}
```

## シングルオブジェクト型

シングルオブジェクト型は、1つだけのデータとして扱います。レスポンスも単一のオブジェクトが返されます。

```json
{
  "data": {
    "id": "1",
    "name": "John Doe",
    "age": 30
  }
}
```
