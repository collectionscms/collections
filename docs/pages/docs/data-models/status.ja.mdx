# 公開ステータス

> データモデル作成時に、公開ステータスを有効にすることができます。設定された公開ステータスに応じて、データ行が API のレスポンスに含めるかが決定されます。

## 公開ステータスのフィールド

デフォルトで `status` フィールドが作成されます。これは、データモデルの `statusField` から参照され、あとから値を変更することも可能です。

## フィールドの値

デフォルトで `draft`, `published`, `archived` が作成されます。これらは、データモデルの `draftValue`, `publishValue`, `archiveValue` から参照され、あとから値を変更することも可能です。

このようなデータ行が保存されていても

| id  |    title |    status |
| :-- | -------: | --------: |
| 1   | article1 |     draft |
| 2   | article2 | published |
| 3   | article3 |   drafted |

実際は、以下のようにレスポンスされます。

```json
{
  "data": [
    {
      "id": 2,
      "title": "article2",
      "status": "published"
    }
  ]
}
```
