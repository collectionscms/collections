# Superfast CMS へようこそ

```
Superfast はプロジェクト名です。正式名称が決まり次第、お知らせいたします。
```

Superfast はあらゆる規模で利用できる Web アプリとネイティブアプリのバックエンドシステムを構築できます。フロントエンドから API を通じてデータ参照や入力を行ったり、直感的に操作できる管理画面から入稿することが可能です。

今後 Superfast が提供するクラウドサービスでは、Wordpress からのデータ移行も提供を予定しています。

---

現在、正式版リリースに向けて Wordpress や Headless CMS を使った Web 制作を行っている方へのヒアリング調査を実施しています。ご気軽にご相談・ご紹介いただけますと幸いです！
