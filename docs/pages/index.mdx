# Superfast CMS へようこそ

<Callout type="warning">
  Superfast はプロジェクト名です。正式名称が決まり次第、お知らせいたします。
</Callout>

Superfast は Web アプリやネイティブアプリのバックエンドシステムを GUI から設計・構築できる Headless CMS です。フロントエンドから API を通じてデータ参照や登録を行ったり、管理画面から直感的に操作しながら入力することが可能です。

今後 Superfast が展開するクラウドサービスでは、Wordpress からのデータ移行も提供を予定しています。

---

現在、正式版リリースに向けて Wordpress や 既成の Headless CMS を使った Web 制作を行っている方へのヒアリング調査を実施しています。ぜひ、ご相談・ご紹介いただけますと幸いです！
